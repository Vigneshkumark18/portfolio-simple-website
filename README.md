This is a basic HTML website with a homepage and contact page, styled using CSS. The project demonstrates how to structure a simple static website while applying basic styling for an enhanced user experience.

**Features**
**Homepage:**
A clean and visually appealing design with introductory content.
Responsive layout elements to provide a user-friendly interface across different devices (if you've added media queries).

**Contact Page:**
A form where users can enter their name, email, and message to reach out.
Styled input fields and buttons for a polished look.

**Includes:**
Text inputs for name and email
Textarea for user messages
Submit button (ready to be integrated with backend functionality)

**Technologies Used**
HTML5: For structuring the content and layout of the website.
CSS3: For adding style, including:

**Layout formatting**
Typography
Button and form styling
Any responsive design techniques (if applicable)
How to Use
Download or clone this repository.
Open the index.html file in your web browser to view the homepage.
Navigate to the contact.html file to check out the contact form and its styling.
Both pages are styled using the linked style.css file.

**Future Enhancements**
Add JavaScript for form validation and interactivity.
Integrate backend to make the contact form functional.
