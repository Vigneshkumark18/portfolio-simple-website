# My portfolio website

This is a simple portfolio website created using HTML and basic CSS, featuring a home page and a contact page.

**Features:**

Clean Design: The website features a minimalist design, ensuring a user-friendly experience.
Responsive Layout: Built with responsiveness in mind, ensuring optimal viewing across various devices and screen sizes.
Project Showcase: The home page showcases my projects, skills, and achievements.
Contact Form: The contact page includes an interactive form allowing visitors to get in touch easily.
Customizable: The code is easy to understand and modify, making it simple to customize the website according to individual preferences.

**Technologies Used:**
HTML
CSS

**How to Use:**
Clone the repository to your local machine.
Open the index.html file in your preferred web browser to view the home page.
Navigate to the contact.html file to view the contact page.
Customize the content and styles as needed to personalize your portfolio.
