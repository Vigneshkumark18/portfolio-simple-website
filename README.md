Personal Portfolio Website  

This repository contains my **personal portfolio website**, built using **HTML** and **CSS**.  
The main goal of this project is to practice web development fundamentals and create a platform where I can showcase my skills, projects, and achievements.  

---

## ✨ Features
- Clean and simple design  
- Home and Contact sections  
- Easy navigation with responsive layout (basic)  
- Showcases my work and personal details  

---

## 🛠️ Tech Stack
- **HTML5** – for structuring the web pages  
- **CSS3** – for styling and layout
