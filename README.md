# portfolio-simple-website

Personal Portfolio Website

This is a **simple and responsive personal portfolio website** built using **HTML** and **basic CSS**. It is designed to showcase my profile, skills, projects, and contact information in a clean and professional layout.

## 📄 Description

This project is my personal portfolio website that includes the following sections:
- **Home**: A brief introduction and welcome.
- **About Me**: Information about my background and interests.
- **Projects**: A list of personal or practice projects.
- **Contact**: Ways to get in touch with me.

This website is a great starting point for anyone who wants to build a basic portfolio using pure HTML and CSS, without any frameworks or JavaScript.

## 🚀 Features

- Responsive Design  
- Clean Layout  
- Easy to Customize  
- Lightweight and Fast  

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Basic styling, no frameworks)
