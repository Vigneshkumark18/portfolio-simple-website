# portfolio-simple-website

This is a simple personal portfolio website designed to showcase my skills and provide a way for others to connect with me. It includes an index page with an overview of my work and a contact page for communication.

Features:
Home Page (Index):
Highlights my personal profile, skills, and a brief summary of my work.

Contact Page:
A simple form or links to reach out to me via email or social media.

Technologies Used:
HTML: For the structure of the website.
CSS: For styling and layout.

How to Use:
1) Open the index.html file in your browser to view the website.
2) Use the navigation to switch between the Home and Contact pages.
